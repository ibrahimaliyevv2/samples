BRIEF DESCRIPTIONS OF SAMPLES:

  • Orange Ball

    Using arrow keys (↑,↓,←,→) of keyboard you can move ball only inside the given box.

    GitHub Link

  • To do list

    It is one of the ways to keep your works done in time.

    GitHub Link

  • Tabs

    By pressing each tab you can get content related to that tab (related text will be displayed below tabs).

    GitHub Link

  • Preview of image

    You can upload image (or drag & drop) and see it on page.

    GitHub Link

  • List to List

    By dragging list item you can either add to second list (or vice versa). Also by dropping it to the green box you can print its content as paragraph inside that box.

    GitHub Link

  • Order subscription

    Simple template for ordering subscription for various platforms like music, etc.

    GitHub Link

  • Make text bigger

    Each time you click the button, font-size of text is getting bigger and bigger.

    GitHub Link

  • Hover to navbar item

    When you hover text you can see yellow color appears after it. This technique is widely used while designing navbars for websites.

    GitHub Link
